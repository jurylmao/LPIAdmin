# LPIAdmin
"moderation" tool for Let's Party Infinite

## Features
- ban (from server)
- kick
- serverlock
- softlock
- get f3x
- teleport to island
- teleport to museum
- delete gearwalls
- delete btools
- delete everything
- kill
- freeze
- joinlogs
- disable chat
